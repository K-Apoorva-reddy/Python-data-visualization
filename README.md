# Data Visualization using Matplotlib

A Python-based data visualization project that reads employee data from a CSV file, performs data cleaning using Pandas, and generates visual insights using Matplotlib. This project was developed as **Task 3** of my Python Internship.

Project Overview

This project demonstrates how to clean and analyze employee data before presenting it through graphical visualizations. The application loads employee records from a CSV file, handles missing values, removes duplicate entries, calculates salary statistics, and generates multiple charts for better understanding of the dataset.

Features

- Read employee data from a CSV file
- Handle missing values
- Remove duplicate records
- Calculate:
  - Average Salary
  - Highest Salary
  - Lowest Salary
- Generate Bar Chart
- Generate Pie Chart
- Generate Histogram
- Save cleaned data to a new CSV file

Technologies Used

- Python 3.x
- Pandas
- Matplotlib
- CSV Files

How to Run

Clone Repository

```bash
git clone https://github.com/K-Apoorva-reddy/Python-data-visualization.git
```

 Install Required Libraries

```bash
pip install pandas matplotlib
```

Run the Program

```bash
python main.py
```

Output

- Displays salary statistics
- Generates:
